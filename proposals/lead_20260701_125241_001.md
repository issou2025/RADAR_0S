# Project Proposal — Bim Modeling

## Project Understanding
You need assistance with: 
**Draft: locked fields do not unlock after entering a point**

*Project description:*
### Problem description

There is an issue with inserting 2nd point of Draft objects, which have the option to insert polar coordinates (Length + Angle) - for example `Line` or `Move`.

I am not able to reproduce with FC 1.1.1. It is a regression.


### Workbench affected?

Draft

### Steps to reproduce

- open new file
- start `Draft_Line`
- insert the first point [0,0,0] using keyboard

### Expected behavior

It should be possible to insert second point.

### Actual behavior

It is not possible to insert second point.

See behavior in the video:
