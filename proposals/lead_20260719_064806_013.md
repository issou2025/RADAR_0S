*Project description:*
Hi, and thanks for maintaining `wyze-sdk`.

I wanted to share a small library/CLI I built on top of this SDK in case it is useful to other users here:

https://github.com/jayzeng/wyze-local-devices

It is a compact Python CLI for discovering, caching, looking up, and controlling Wyze devices locally from the command line. The goal is not to replace `wyze-sdk`, but to provide a thin convenience layer for common automation workflows.

A few things it currently supports:

- List Wyze devices as a table or stable JSON
- Cache discovery results in SQLite for faster local lookup
- Look up devices by nickname, MAC, type, or model
- Turn matching plugs, bulbs, and cameras on/off
- Set bulb brightness, color temperature, and RGB color
- Adjust bulb brightness relatively, like `brighter` or `dimmer`
- Verify plug/bulb state after commands when the SDK exposes state checks
- Run as a standalone `uv` script or install as a CLI
- Bundle an optional Codex/OpenAI agent skill for local smart-home workflows

Example usage:

```bash
uv run --script wyze_devices.py list --all --json
uv run --script wyze_devices.py lookup camera
uv run --script wyze_devices.py control "desk plug" off --verify --json
uv run --script wyze_devices.py adjust-light "corner" brighter --step 20 --verify
uv run --script wyze_devices.py set-light "corner" --color ff8800 --verify --json
```
