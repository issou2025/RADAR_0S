# Project Proposal — Flutter App

## Project Understanding
You need assistance with: 
**[feature request] Add Dart syntax highlighting support**

*Project description:*
### Summary

Please add support for **Dart** syntax highlighting in code blocks.

### Motivation

Docmost already supports syntax highlighting for many programming languages, but Dart is currently missing. This makes it difficult to use Docmost as a knowledge base for Flutter and Dart development.

Dart is the primary language for Flutter, which has a large and active developer community. Adding Dart would improve the experience for developers documenting Flutter projects, code snippets, and technical notes.

### Proposed solution

Include the Dart language definition from `highlight.js` and make it available in the list of supported code block languages.

Since `highlight.js` already provides a Dart language definition, I believe this may only require registering the language and exposing it in the editor.

### Example

````markdown
```dart
void main() {
  print('Hello, World!');
}
```
````

### Additional context

I am currently evaluating Docmost as a replacement for Obsidian. Lack of Dart syntax highlighting is one of the few missing features I have encountered so far. Adding support for Dart would make Docmost a much better fit for Flutter developers.

I'd be happy to test the implementation if needed.
