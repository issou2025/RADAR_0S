# Project Proposal — Flutter App

## Project Understanding
You need assistance with: 
**spike: macOS-native MobiSSH — feasibility, plugin/FFI audit, phased plan (Mac on agent-hub)**

*Project description:*
Owner question (2026-07-08): what would it take to build a macOS-compatible native MobiSSH? A Mac exists on the agent-hub fleet ("macbook") for building/testing.

## Spike deliverable
A concrete feasibility report + phased plan covering:
1. **Flutter macOS target**: enable the macos/ platform, what of the app tree is platform-coupled.
2. **Plugin audit** (pubspec): flutter_foreground_task (Android-only — the keepalive/session-host isolate architecture is built on it; macOS needs a different lifecycle story, likely simpler: no FGS needed, plain background isolate or none), flutter_secure_storage_macos, local_auth_darwin, flutter_local_notifications (macOS support), file_selector (macOS ok), share_plus, url_launcher, webview_flutter (macOS = WKWebView via webview_flutter_wkwebview?), device_info_plus, path_provider, shared_preferences — flag each as ok / needs-variant / needs-rework.
3. **flterm/libghostty FFI**: the biggest unknown — does the libghostty pub package (pinned 0.0.9) ship macOS dylibs / build from source on macOS? How does the vendored fork build its native lib per platform?
4. **Architecture seams**: task-isolate/IPC (session_host, task_ssh_gateway, keepalive_task) — which parts are Android-shaped (FGS notifications, battery optimization, POST_NOTIFICATIONS) and what the macOS equivalent is (probably: no keepalive needed, direct in-process host).
5. **Input/UX deltas**: physical keyboard first (the mobile keybar/IME plumbing becomes optional), window resize vs keyboard-inset logic, menu bar, Cmd-shortcuts.
6. **Build/distribution**: must build ON a Mac (agent-hub `macbook`) — enrollment status, toolchain needs (Xcode, flutter), signing/notarization options for personal use (ad-hoc / developer-id), artifact publishing alongside the APK pipeline.
7. **Phasing**: bot-sized slices with a realistic first milestone (e.g. "connects + renders a session with Ghostty on macOS, no keepalive, no biometrics").
