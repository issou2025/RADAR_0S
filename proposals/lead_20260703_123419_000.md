## Project Understanding
You need assistance with: 
**Enhancement Proposal: IFC Property Unit Display & Conversion**

*Project description:*
### The problem

Currently, values are displayed without units in certain properties, such as volumetric flow rate.

[VZT.ifc.txt](https://github.com/user-attachments/files/29633071/VZT.ifc.txt)

<img width="1188" height="423" alt="Image" src="https://github.com/user-attachments/assets/53fe9ae9-f19a-4197-9a59-340e77e04213" />

### Proposed solution

First Proposal: Add unit visibility here, similar to how it is already handled for area ($m^2$) in quantity sets. For instance, in my HVAC use case, this would apply to IfcVolumetricFlowRateMeasure.

Second Proposal: Introduce a tool to convert default IFC units to specific user needs. For example, the default unit for IfcVolumetricFlowRateMeasure is $m^3/s$ (which, to my best knowledge, is the only format Revit exports). However, in practice, I need $m^3/h$. It would be highly beneficial if IFClite allowed users to modify this. In this specific case, the conversion is just a simple multiplication by 3600 with some rounding.

Thank you for considering this solution :)
