## Description

**Background.** The Dev/demo-mode switch and any future experiment flags currently sit in the normal Settings screen.

**Reason for change.** Moving them into the hidden Developer menu keeps everyday Settings clean for a normal user, with all developer toggles in one place.

## Acceptance Criteria

- [ ] Dev/demo-mode toggle moved into the Developer menu

- Add/extend an integration test in demo mode under `integration_test/` (pumpDemoApp harness): consumer Settings has no dev toggles; the Developer menu does.
